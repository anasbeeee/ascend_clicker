# ASCEND

ASCEND est un jeu de clicker narratif jouable sur navigateur, centré sur la progression, l’automatisation et la construction d’un univers original.

Le joueur commence par une interaction simple (cliquer), puis débloque progressivement des mécaniques plus complexes : automatisation, stratégie, recrutement de personnages et évolution à long terme.

---

## Concept du jeu

ASCEND est un **clicker narratif mi-idle**, dans lequel :
- le clic est essentiel au début,
- l’automatisation prend ensuite le relais,
- la stratégie devient centrale à long terme.

Le jeu s’inspire de l’approche expérimentale de projets comme *neal.fun*, tout en développant **son propre univers**, son lore et ses personnages.

---

## Gameplay principal

- Cliquer (bouton et barre espace)
- Accumuler des ressources
- Acheter des améliorations
- Débloquer de nouvelles mécaniques
- Recruter des personnages via des missions
- Construire une équipe (approche proche d’un système de collection)
- Progresser sur le long terme grâce à des systèmes passifs
- Rebirth après la fin de l’histoire pour relancer une progression infinie

---

## Narration et progression

ASCEND possède :
- une **histoire complète avec une fin**,
- un univers évolutif,
- des personnages jouables ou recrutables.

Même après la fin de l’histoire, le jeu reste jouable grâce au système de rebirth et à la progression infinie.

## Technologies utilisées

- 
- CSS (simple et lisible)HTML
- JavaScript 

Aucun framework n’est utilisé volontairement afin de :
- mieux comprendre les bases du développement web,
- garder un projet léger,
- rester totalement maître du code.

Le jeu est conçu pour être :
- jouable sur navigateur,
- responsive,
- hébergé via GitHub Pages.
