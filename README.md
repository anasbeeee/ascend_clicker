# Ascend Clicker

Un jeu de type clicker basé sur la progression, l’ascension et l’optimisation.

## Objectif du projet
Créer un clicker jouable dans le navigateur, avec une vraie sensation de progression
et une structure claire de projet.

## Fonctionnalités prévues
- Système de clic
- Score et ressources
- Améliorations
- Système d’ascension
- Sauvegarde locale

## État du projet
Projet en cours de conception.

## Technologies utilisées
- HTML
- CSS
- JavaScript
